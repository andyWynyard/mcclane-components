## Header

This is a super basic `Header` component that can be used and extended using `styled-components`.
Its highly recomended to extend the `Header` component by using the `styled-component` syntax:

```
const YourHeader = styled(Header)<HeaderProps>`
  property: ${props => props.dynamicProp};
  property2: your css here;
`
```

The `props.children` can be anything, but is essentially designed to be used with text.
