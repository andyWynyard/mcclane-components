# Typography Body

To use this component, simply pull it like like a so.

`import Typography from 'mcclane-components'`
`const { H1, H2, H3, H4, H5, LeadSmall, LeadLarge, Paragraph, ParagraphSmall } from Typography`

This system uses two libraries to build itself. [styled-components](https://styled-components.com/) and [styled-system](https://styled-system.com/)

This componend _extends_ the `<GenericComponent />` which incorporates all of the props that are exposed via the `styled-system` library. The props tables below are as such not exhaustive.

Each of the props that can be used from `styled-system` are from a specific module. The reference for each module can be found here:

- [space](https://styled-system.com/table#space)
- [typography](https://styled-system.com/table#typography)
- [color](https://styled-system.com/table#color)
- [layout](https://styled-system.com/table#layout)
- [flexbox](https://styled-system.com/table#flexbox)
- [grid](https://styled-system.com/table#grid)
- [background](https://styled-system.com/table#background)
- [border](https://styled-system.com/table#border)
- [position](https://styled-system.com/table#position)
- [shadow](https://styled-system.com/table#shadow)
